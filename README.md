# Bookmark My Food
Bookmarking platform to save and organize your must try restaurants

## Table of Content
1. Set-up 
1. Screenshots

## Set-Up
- Navigate to client folder:
`bower install`
- Navigate to server folder
`npm install`
- Launch server `node server.js`
